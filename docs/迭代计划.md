## 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| Tushare 接口不稳定 | 数据获取失败 | 预留 akshare 作为备选 |
| 评分算法效果不佳 | 决策质量低 | 先用简单规则，后续迭代优化 |
| 账户数据手工维护繁琐 | 更新不及时 | 后续迭代考虑 API 接入 |
| 前端开发进度慢 | 整体延期 | V1.0 可先用 API + JSON 输出 |
