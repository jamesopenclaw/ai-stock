# 轻舟版交易系统

A 股短线交易决策支持系统

## 项目简介

轻舟版交易系统是一套面向 A 股短线交易场景的决策支持系统，围绕"市场环境判断—板块扫描—个股筛选—买点分析—卖点分析—账户适配—执行输出"构建完整的交易决策链路。

## 核心功能

- 市场环境分析
- 板块扫描
- 个股筛选
- 买点分析
- 卖点分析
- 账户适配
- 风控提醒
- 结构化输出
